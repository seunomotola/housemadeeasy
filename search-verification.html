How to Use:

Place all PHP files in your web server directory
Ensure database connection details in inc/connect.inc.php are correct
Open search-made-easy-new.php in your browser
Use the search form to filter properties by location and type
Click on properties to view details with video functionality

Search Functionality Details:

Search Parameters:
- Location: Sagamu (fixed option)
- Type: Single Room, Self Contain, 1 Bedroom Flat, 2 Bedroom Flat, 3 Bedroom Flat, 4 Bedroom Flat

When you search:
- If both filters are selected, it searches for properties matching both
- If only one filter is selected, it searches for properties matching that filter
- If no filters are selected, it displays hot properties (house_label='hot')

Search Results Display:
- Properties are shown in a responsive grid
- Each property shows location, price, rooms left, negotiable status
- Booked/unavailable properties have a status indicator
- Click "View Details" to see complete information

Video Functionality:

If a property has a YouTube video:
- A video button appears in the bottom-right corner of the main image
- Clicking the button opens a modal with the YouTube video
- The modal can be closed with the X button or by clicking outside
- ESC key also closes the modal
- Video starts playing automatically

Video Button Style:
- Rectangular with 10px border radius
- Contains play icon and "Watch Video Tour" text
- Hover effect (slightly darker background, subtle transform)
